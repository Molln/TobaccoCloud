路由转发

网关路由转发

1 Nacos 新建配置

  • Data ID

    tobacco-gateway-router

  • Group

    DEFAULT_GROUP

  • 配置格式

    JSON

  • 配置内容

    [
    {
        "id": "demo",
        "order": 0,
        "predicates": [{
            "args": {
                "pattern": "/demo/**"
            },
            "name": "Path"
        }],
        "uri": "lb://demo-service"
    }
]

参数说明

参数 说明
id 路由标识，具有唯一性
order 排序
predicates.args.pattern 路由转发条件
predicates.name 固定格式
uri 目标服务地址，lb 表示负载，后面加 Nacos 中服务名
